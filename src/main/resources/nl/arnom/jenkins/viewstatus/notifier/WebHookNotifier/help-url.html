The url will be called with a HTTP POST request containing a JSON object in the body similar to:

{
  "view": {
    "name": "MyView",
    "display_name": "My View",
    "url": "http://localhost:8080/view/MyView/"
  },
  "result": {
    "text": "SUCCESS",
    "numeric": 0
  },
  "old_result": {
    "text": "FAILURE",
    "numeric": 2
  },
  "startup": false
}

Note that old_result is optional and might not be included. During the start up phase of Jenkins CI the current status is determined. If the web hook is called because of this startup will be set to true.
